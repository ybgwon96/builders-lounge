# Builder's Lounge (빌라) 랜딩페이지 디자인

## 개요
Build in Public 하는 한국 쓰레더들의 모임 "빌라(Builder's Lounge)"의 랜딩페이지.
멤버들의 프로젝트 쇼케이스가 주요 목적.

## 세계관 & 비주얼 컨셉

페이지 전체가 하나의 건축 세계관. 스크롤 여정: **빌라(현재) -> 건설 현장(과정) -> 아파트 단지(목표)**

- **색감:** 다크 네이비 하늘 배경 + 따뜻한 창문 불빛(노란/주황) + 네온 악센트
- **일러스트:** CSS/SVG 기반 건물 실루엣, 크레인, 건설 현장 요소
- **애니메이션:** 스크롤에 따라 건물이 올라가고 불이 켜지는 인터랙션
- **핵심 메타포:** 개별 빌라가 아니라 하나의 큰 아파트를 함께 짓는다 = 커뮤니티의 집단 성장

## 섹션 구성

### 1. 히어로 - "지금은 빌라, 목표는 아파트"
- 작은 빌라 한 채에서 시작, 스크롤하면 크레인과 함께 고층 아파트가 올라옴
- 카피: "Build in Public 하는 한국 쓰레더들의 모임"
- 부제: "함께 짓고, 함께 올라갑니다"

### 2. 프로젝트 쇼케이스 - "우리가 올리고 있는 층수"
- 각 프로젝트가 아파트 한 세대(호수)로 표현
- 건물 전체가 하나의 아파트이고, 프로젝트가 쌓일수록 층이 높아지는 구조
- 멤버가 많아질수록 아파트가 커지는 느낌 = 커뮤니티 성장의 시각화
- 카드: 프로젝트명, 빌더 이름, 쓰레드 링크, 간단 설명

### 3. CTA - "입주 신청"
- 완성된 고층 아파트 + 빈 세대 하나에 불이 깜빡이는 연출
- 카피: "당신의 호수가 비어 있습니다"
- 카카오톡 오픈채팅 링크 버튼

## 데이터
- Supabase DB에서 프로젝트 데이터 관리
- 테이블: projects (프로젝트명, 빌더명, 설명, 쓰레드 링크, 이미지 등)

## 기술 스택
- Next.js + Tailwind CSS + Framer Motion (스크롤 애니메이션)
- Supabase (프로젝트 데이터)
- Vercel 배포
